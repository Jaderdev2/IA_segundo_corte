Este repositorio contiene el desarrollo de un taller investigativo sobre la librería **Pandas en Python**, realizado en Google Colab.  
El trabajo incluye una investigación teórica y ejercicios prácticos orientados al análisis y manipulación de datos.

## 🎯 Objetivo

Comprender el uso de la librería Pandas y su aplicación en el análisis de datos, ciencia de datos e inteligencia artificial, mediante ejercicios prácticos en Python.

---

## 📖 Temas investigados

| Tema | Breve descripción |
|------|-------------------|
| **¿Qué es Pandas?** | Librería Python para manipulación eficiente de datos tabulares. |
| **¿Para qué sirve?** | Limpiar, transformar, organizar y analizar datos estructurados. |
| **Objetivo principal** | Proveer estructuras rápidas (DataFrame, Series) para trabajar con datos etiquetados. |
| **Principales funciones** | `read_csv`, `head`, `info`, `describe`, `groupby`, `sort_values`, etc. |
| **Importancia en IA** | Base para el análisis exploratorio (EDA) y preparación de datos en Machine Learning. |
